# CS-370-Current-Emerging-Trends
On this project i showcased the development of a Pirate Intelligent Agent using reinforcement learning and neural networks. The goal was to design an agent capable of learning optimal decision-making strategies within a simulated environment. The completed Jupyter Notebook and exported HTML file demonstrate how key artificial intelligence concepts were applied to solve a real-world inspired problem.Through this project, I implemented and trained a reinforcement learning model that enables the pirate agent to explore its environment, learn from rewards and penalties, and progressively improve its performance over time.
For the Code Provided:
The starter code included:

The environment setup for the pirate simulation

Initial class structures and helper functions

Predefined parameters for training

A general reinforcement learning framework

This provided the structure for interacting with the environment, managing states, and defining reward mechanics.

And for the Code I Created,

I developed and implemented:

The reinforcement learning logic for the pirate agent

Q-learning or neural network-based decision-making components

The training loop for iterative learning

Exploration vs. exploitation strategies

Reward optimization adjustments

Performance tracking and evaluation

I configured hyperparameters, adjusted learning rates, and refined reward mechanisms to ensure the agent learned efficiently and converged toward optimal behavior.
